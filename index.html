How to Share Your Design

  1. Fork this repository
  2. Add your design image to the designs/images/ folder
  3. Add your design JSON file to the designs/data/ folder
  4. Update the designs/index.json file with your design entry
  5. Create a pull request

File naming: Use format your-design-name.jpg and your-design-name.json
